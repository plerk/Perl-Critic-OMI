# Perl::Critic::OMI

Perl::Critic policies for OMI

# SYNOPSIS

    % perlcritic --quiet --theme omi .

# DESCRIPTION

Provide some special case [Perl::Critic](https://metacpan.org/pod/Perl::Critic) policies used by OMI.
